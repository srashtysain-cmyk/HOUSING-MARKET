
Project Documentation
Visualizing Housing Market Trends

1. Project Objective
The objective of this project is to analyze housing market data and understand the factors that affect house prices and sales trends. The project uses data visualization to help users easily understand market patterns and make better decisions.

2. Project Description
This project analyzes housing data and shows the results using charts and dashboards. It helps users understand trends in house prices, renovations, and property features.
Technologies Used
•	Python (Flask)
•	Tableau
•	HTML, CSS, JavaScript
Target Users
•	Real estate analysts
•	Marketing teams
•	Company managers
________________________________________
3. Application Scenarios
Overall Data Overview
Shows the total number of houses in the dataset and the average house price.
Sales by Renovation Years
Shows how renovation affects house prices and sales.
House Age Distribution
Displays the age of houses and renovation status.
Property Features Analysis
Compares house features such as bedrooms, bathrooms, and floors.

6. System Workflow
1.	User opens the project website.
2.	User selects filters like house age or renovation status.
3.	Tableau processes the data.
4.	Flask manages the backend.
5.	Charts update automatically based on the selected filters.
